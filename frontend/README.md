# SGSMM Frontend Dashboard

Next.js 16 dashboard for the Sortino-Gated Smart Money Mirror. Displays vault overview, smart-money leaderboard, decision feed, and equity curve proof chart.

## Quick Start

### Prerequisites
- Node.js ≥ 22.0.0
- pnpm (or npm/yarn)

### Development

From repo root:

```bash
pnpm frontend:dev
```

Or directly from `frontend/`:

```bash
cd frontend
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build & Run

```bash
cd frontend
pnpm build
pnpm start
```

Production build outputs to `.next/`; `pnpm start` serves it locally on port 3000.

## Data Source

The dashboard reads **backtest results** via static JSON files generated by the Python backtest infrastructure:

- **Leaderboard:** `public/data/leaderboard.json` (wallets ranked by Sortino)
- **Equity curve:** `public/data/equity_curve.json` (nav over 26 epochs)
- **Decision feed:** `public/data/decisions.json` (every ENTER/HOLD/DEFUND action)
- **Metrics:** `public/data/summary.json` (total return, max drawdown, Sortino, passes_kill_criterion)

These files are **generated by `scripts/build_data.py`** from the backtest run results (`../backtest/results/`).

### Regenerate Dashboard Data

When you run the Python backtest, regenerate the dashboard data:

```bash
cd backtest
.venv\Scripts\python.exe scripts/build_data.py
```

This updates all files in `frontend/public/data/` so the dashboard reflects the latest backtest snapshot.

## Architecture

### Key Pages

- **`app/page.tsx`** — Main dashboard: vault NAV, key metrics, Sortino gate status
- **`app/leaderboard/page.tsx`** — Smart-money wallet ranking by Sortino
- **`app/decisions/page.tsx`** — Historical feed of all agent decisions
- **`app/proof/page.tsx`** — Equity curve chart: SGSMM vs naive baseline

### Components

- **`components/VaultCard.tsx`** — Vault overview (deposit/withdraw UI placeholders)
- **`components/EquityCurve.tsx`** — Recharts visualization of equity over time
- **`components/LeaderboardTable.tsx`** — Sortable wallet table
- **`components/MetricsSummary.tsx`** — Sortino, max drawdown, passes_gate display

### Styling

- **Tailwind CSS v4** for styling
- **Recharts** for charts
- Dark mode via `prefers-color-scheme` media query
